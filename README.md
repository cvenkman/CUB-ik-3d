## cub3d

with [Vova](https://github.com/delakruel/) :)

Raycasting game (almost a game)

### Compiling

* To compile
	- `make`
* To remove objects:
	- `make clean`
* To remove objects and binary file (program):
	- `make fclean`
* To re-compile:
	- `make re`

### Executing

`./cub3d maps/test.cub`

* ``WASD`` to move, ``Left/Right`` directional keys to rotate camera
* ``ESC`` to quit
